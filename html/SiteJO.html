Description

Réalisation d'un site web sur les JO de Paris 2024 en HTML et CSS.
Pour commencer, nous avons analysé une interview portant sur les besoins d’un client. À partir de ces besoins, nous avons créé une charte graphique décrivant les polices d'écriture utilisées, les couleurs ainsi que quelques mises en situation et présentations de contraste. Cette charte graphique nous a ensuite permis de créer une maquette du site web sur Figma, contenant une page d'accueil, une page sur une discipline présente aux JO de Paris 2024, une page sur une des épreuves de la discipline choisie et une page sur un sportif ou une sportive aillant participé à l’épreuve choisie. Enfin, nous avons pu réaliser le site lui-même en nous organisant via GitHub.
Dans ce projet, l'analyse des besoins et la charte graphique ont été réalisées par tout le groupe, mais pour le reste, chacun était responsable d'une page. Pour ma part, je travaillais sur celle concernant la sportive.

Etapes

  2. Réalisation d'une charte graphique
  3. Réalisation d'une maquette
